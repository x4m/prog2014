План практики по LINQ
1. Вывести список Customers
2. Изменить форматирование списка Customers
3. Вывести список Customers, для каждого Customer вывести список его Orders
4. Вывести список всех Products, которых на складе осталось меньше 3 (Where)
5. Вывести список всех Customers из штата Вашингтон ("WA") (Where)
6. Для всех цифр текстом
string[] digits = { "zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine" };
найти цифры, число букв в которых меньше их значения. (Where)
7. Для всех цифр текстом создатьперечисление названия цифр с большой буквы и перечисление цифр капсом. (Select)
8. Найти минимальную цену продукта (Select,Min)
9. Найти продукт с максимальной ценой (Select,OrderByDesc,First)
10. Вывести список Заказчиков из штата Вашингтон посчитать общую сумму заказа (Where,SelectMany,Sum)
11. Найти пять заказчиков с максимальной суммой одноразового заказа (Select,OrderBy,Take)
12. Найти пять заказчиков с максильмной суммой всех заказаов
13. Найти тех заказчиов,которые вошли в оба списока (Intersect)
14. Найти объединение этих двухсписоков (Union)
15. Найти конкатенацию эттих двух список (Concat)
16. Сгруппировать заказчиок по первой букве имени и найти сумму заказов на каждую букву(GroupBy,Select,Sum)
